You Cant C Me

DESCRIPTION : Can you see me?

Basic Info
File command

It is 64 Bit elf LSB binary

Strings command

There is some interesting strings.

  • this_is_the_password
  • HTB{%s}
  • I said, you can't c me!
Run the binary
  • It print Welcome! strings
  • Then ask for some input
  • Then it print I said, you can't c me!
Ltrace commnad
  • There is a strcmp library call
  • It compare the two strings
  • First one is the flag and the second one is our input
  • Let's check the value
Check the value

Yeah!!! The flag is correct!.
